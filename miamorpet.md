# Mi Amor Pet – Requerimientos

## Contexto
Emprendimiento paralelo solicitado por Dafna y Doraluz durante la reunión del 25/11. Enfoque en accesorios de lujo para mascotas con posicionamiento premium.

## Necesidades clave
1. **Carrito e-commerce inteligente**
   - Experiencia de compra fluida para productos personalizados de mascotas.
   - Integración con Shopify o storefront propio que pueda embebirse en la misma PWA.

2. **Personalización de productos**
   - Selección de materiales (cuero, metal) y opción de grabado con nombre/mensajes.
   - Registrar especificaciones (tamaño, color, tipo de mascota) y previsualizar.

3. **Contenido generado con IA**
   - Herramienta para crear mockups/UGC de perros y gatos usando IA, reutilizable en campañas.
   - Posible flujo: subida de fotos → IA genera variantes con el accesorio personalizado.

4. **Roadmap sugerido**
   - **Fase 1**: Carrito + catálogo + configurador de grabado.
   - **Fase 2**: Motor de recomendaciones inteligentes (up-sell de correas, placas, etc.).
   - **Fase 3**: Generación automática de contenido (IA) + panel de assets descargables.

5. **Dependencias**
   - Definir inventario inicial y SKUs.
   - Integrar pasarela de pago compatible con Shopify / PWA.
   - Requisitos legales para grabado personalizado (tiempos de producción, devoluciones).
