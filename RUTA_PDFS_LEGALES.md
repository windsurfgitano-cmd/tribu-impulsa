# Ruta para PDFs Legales

Cuando conviertas los archivos Word a PDF, colócalos en la siguiente ubicación:

**Ruta**: `public/legales/`

**Nombres de archivos**:
- `terminos-y-condiciones.pdf`
- `politicas-de-privacidad.pdf`
- `faq.pdf` (si aplica)

Luego, estos PDFs podrán ser referenciados desde el código para descarga o visualización.
